Class Memories

Separate pages make updates easier

Each class page will have its own colour theme, welcome section, school year area, photo space and class update section.

This makes the website easier to manage. When Green Class sends photos, only the Green Class page needs to be updated. When Blue Class sends photos, only the Blue Class page needs to be updated.

Class pages can grow naturally over time with approved school updates and memories.

Future Updates

Approved photos and class news

Class pages can be updated with approved photographs, class activities, school events and short pieces of information supplied by the school.

For pupil safety and privacy, no child names should be published with photographs unless this has been approved by the school.

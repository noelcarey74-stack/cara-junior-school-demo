Class Memories

Separate pages make updates easier

Each class page has its own colour theme, welcome area, class team section, photo space and class update section.

This makes the website easier to manage. When one class sends an update, only that class page needs to be changed.

The class pages can grow naturally over time with approved school updates, photographs and memories.

Privacy & Care

Approved photos and class news

Class pages can be updated with approved photographs, classroom activities, school events and short pieces of information supplied by the school.

  • Photographs should be approved by the school before being added.
  • Pupil names should not be attached to photographs unless approved by the school.
  • Teacher and SNA names can be added once confirmed and approved.

How this page will grow

When Green Class sends approved photos and text, this page can be updated with a new memory card.

Each update should include a title, date, short description, class name and confirmation that the photos are approved for the website.

For pupil privacy, children’s names should not be published with photographs unless this has been specifically approved by the school.

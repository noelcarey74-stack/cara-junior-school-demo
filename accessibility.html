Website Accessibility

Accessibility Statement

Cara Junior School aims to make this website clear, accessible and easy to use for parents, guardians, staff, visitors and the wider school community.

Accessible Information

This website is designed to be simple, readable and easy to navigate.

Accessibility Statement

Our accessibility approach

Cara Junior School wants this website to be useful and accessible to as many people as possible. The website has been designed with clear navigation, readable text, strong contrast and simple page layouts.

What we aim to provide

The website aims to support visitors by using clear headings, descriptive links, readable font sizes, meaningful image descriptions and a consistent layout across pages.

Keyboard navigation

The website includes a skip link at the top of each page so keyboard users can move directly to the main content. Links and buttons are intended to be usable by keyboard as well as mouse or touch.

Images and text

Images used on the website should include suitable alternative text where possible. School photographs and class images should only be uploaded where approved by the school.

Clear language

Website pages are written in plain, clear language where possible. Policy pages are presented as readable webpages to make school information easier to access.

External documents

Some school documents may also be available as PDF files. Older PDF documents may not always be fully accessible. Where possible, important information is also presented as a clear webpage.

Known limitations

This website is a demo website prepared for review. Some content, images, policies or documents may need to be checked and approved by the school before final publication.

Feedback

If you have difficulty accessing information on this website, please contact the school office so the issue can be reviewed.

Accessibility Statement

Our commitment to accessibility

Cara Junior School wants this website to be easy to use and accessible to as many people as possible.

The website has been designed with clear navigation, readable text, strong contrast, mobile-friendly layouts and simple page structures.

Website features

This website includes:

  • Clear page headings and structured content.
  • A skip-to-content link for keyboard users.
  • Readable font sizes and strong colour contrast.
  • Mobile-friendly layouts for phones and tablets.
  • Alternative text for important images.
  • Simple navigation and consistent page layouts.

PDF documents

Some school information may be provided as PDF documents. The school aims to present these documents in a clear policy library so they are easier to find and download.

If you have difficulty accessing a PDF or any information on this website, please contact the school office.

Feedback

If you find any part of this website difficult to use, or if you need information in another format, please contact the school office.

This page is for Cara Junior School staff to send approved website updates, class news, photos, policy updates and school information.

Important

Only approved information and approved photographs should be submitted for the website.

Before You Send

Photo and privacy reminder

Please only send photographs and information that have been approved by the school.

Do not include pupil names with photographs unless this has been specifically approved by the school.

Please include the class colour, event title, date and a short description with each update.

Send Update

Choose the update type

Select a class or update type below. A prepared email will open with the correct subject and template.

How It Works

Email template system

When a staff member clicks a button, the page will ask for a simple staff code. If the code is accepted, their email app opens with a prepared website update email.

This keeps the public website simple and avoids a complicated backend or login system.
